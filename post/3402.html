倒装

发表于2023-01-03|更新于2023-02-11|英语
|字数总计:615|阅读时长:2分钟|阅读量:

什么是倒装？

倒装是指将句子谓语放在主语之前，常常具有强调语气。

部分倒装

把 be 动词、助动词或情态动词 置于主语之前的句子（提前）叫部分倒装句。

1）否定词在句首

常见否定词：seldom, hardly, little, few, neither, nor, scarcely, nowhere, no longer, not unitl, no soon … than …, hardly … when …, not once, in no way, in no case, at no time

2）so/such … that中的 so/such在句首

so + adj/adv 或 such + n在句首, 使用部分倒装

例句：

Linda was so proud that she never listened to any advice.
So proud was Linda that she never listened to any advice.（is提前）

3）hardly / scarcely / barely … when / no sooner … than

主句 用 过去完成时（no sooner/hardly），从句 用 一般过去时。

4）让步状语从句（与众不同）

让步状语从句何时用半倒装

  1. as 引导的让步状语从句 必须使用半倒装

  2. though 引导的让步状语从句，可半倒装，也可不倒装（随意）

  3. although引导的让步状语从句，不可倒装

  4. no matter how / however 引导的让步状语从句，使用半倒装

公式：

  1. as/though：形容词/分词/动词原型/副词/名词 + as/though + 主 + 谓

  2. no matter how：no matter how + 形容词/副词 +陈述句

5）省略if的虚拟语气

将虚拟语气的if省略，将助动词提至主语前

6）only + 状语 在句首

状语：状语从句/副词/介词短语/时间名词/非谓语动词…

注意：主句倒装；only + 名词/代词 不倒装

7）so/nor/neither的回复

1）so + 助动词 + 主语

要求：
1.回复的上一句为 肯定句
2.两个句子 主语不同，不是同一人
3.两个句子 助动词一致

2）neither/nor + 助动词 + 主语

要求：
1.回复的上一句为 否定句
2.两个句子 主语不同，不是同一人
3.两个句子 助动词一致

8）not only … but also …在句首

not only后面用半倒装，bat also 不变

完全倒装

谓语部分 完全放在主语之前的句子（提前）,便是完全倒装句。

1. there be

2. here/now/then在句首+不及物动词+名词

3. out/in/up/down/away在句首+不及物动词+名词

4. 介词短语（表地点）在句首+不及物动词+名词

5. such + be + 主语

注意：主语为人称代词可以不倒装；完全倒装都是地点在句首

总结
